Resumo da Disciplina

A disciplina tem como objetivo capacitar o aluno no desenvolvimento de aplicações modernas para web e dispositivos móveis, com foco na utilização do React.js para construção de interfaces responsivas, dinâmicas e modulares. Durante o curso, os estudantes aprendem a manipular dados por meio do consumo de APIs REST, utilizar rotas com React Router e aplicar boas práticas no desenvolvimento front-end. Tecnologias como HTML5, CSS3, JavaScript ES6+ e bibliotecas de estilização também são abordadas para garantir uma base sólida no ecossistema web moderno.

Conteúdos Desenvolvidos

  • Fundamentos de HTML5 e CSS3
  • Sintaxe moderna de JavaScript (ES6+)
  • Introdução ao React.js e seu ecossistema
  • Criação de componentes funcionais e gerenciamento de estado
  • Utilização de hooks essenciais: useState, useEffect
  • Navegação de páginas com React Router
  • Consumo de APIs REST com Axios ou Fetch
  • Estilização com CSS-in-JS (Styled-components, Tailwind CSS)
  • Boas práticas no desenvolvimento front-end
  • Controle de versão com Git e GitHub

Abordagem Metodológica

A disciplina é conduzida com forte ênfase prática e desenvolvimento progressivo de projetos. As principais estratégias incluem:

  • Aulas práticas em laboratório com orientação direta do docente
  • Criação de projetos reais utilizando React.js
  • Exercícios individuais e em equipe com foco em resolução de problemas
  • Desenvolvimento de interfaces completas e interativas
  • Avaliações baseadas em desafios práticos com uso de APIs e rotas

Competências Desenvolvidas

Ao término da disciplina, o aluno estará apto a:

  • Desenvolver interfaces web responsivas e de alta performance com React.js
  • Estruturar aplicações front-end utilizando componentização e hooks
  • Realizar requisições a APIs externas para integração de dados
  • Aplicar padrões modernos de estilização com bibliotecas CSS-in-JS
  • Utilizar versionamento de código de forma profissional com Git e GitHub
  • Projetar aplicações web escaláveis, organizadas e com foco na experiência do usuário
