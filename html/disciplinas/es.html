Resumo da Disciplina

A disciplina explora as principais estruturas de dados utilizadas na programação, com foco na implementação prática utilizando a linguagem C. O conteúdo vai desde os conceitos fundamentais, como vetores e matrizes, até estruturas mais sofisticadas, como listas encadeadas, pilhas, filas e filas circulares. Além disso, são trabalhados aspectos essenciais como o uso de structs, ponteiros, alocação dinâmica de memória e modularização com funções — formando uma base robusta para o desenvolvimento de algoritmos eficientes e soluções computacionais bem estruturadas.

Conteúdos Desenvolvidos

  • Revisão dos conceitos fundamentais da linguagem C
  • Manipulação de vetores e matrizes
  • Criação e uso de structs (tipos de dados compostos)
  • Utilização de ponteiros e memória dinâmica
  • Modularização com funções e organização do código
  • Implementação de listas encadeadas (simples e duplas)
  • Pilhas (estrutura LIFO) e suas principais operações
  • Filas (FIFO) e filas circulares
  • Aplicação prática de cada estrutura de dados
  • Análise de eficiência e complexidade dos algoritmos

Exemplo de estrutura de nó para lista encadeada:

typedef struct no {
  int valor;
  struct no *proximo;
} No;

No* criar_no(int valor) {
  No *novo = (No*)malloc(sizeof(No));
  if (novo != NULL) {
    novo->valor = valor;
    novo->proximo = NULL;
  }
  return novo;
}

Abordagem Metodológica

As aulas seguem um modelo teórico-prático, com explanações conceituais seguidas de atividades práticas de codificação. Os estudantes desenvolvem programas em linguagem C aplicando as estruturas estudadas em cenários que simulam situações reais da programação.

As atividades incluem:

  • Exercícios individuais e em grupo
  • Projetos de codificação aplicados a problemas do cotidiano computacional
  • Análise e depuração de código-fonte
  • Avaliações com provas teóricas e implementações práticas

O ambiente de desenvolvimento utilizado foi o Code::Blocks ou Dev-C++, com foco em boas práticas de programação e uso intensivo de recursos como debug e modularização.

Exemplo de implementação de pilha:

typedef struct pilha {
  No *topo;
} Pilha;

void empilhar(Pilha *p, int valor) {
  No *novo = criar_no(valor);
  if (novo != NULL) {
    novo->proximo = p->topo;
    p->topo = novo;
  }
}

Conhecimentos Adquiridos

Durante a disciplina, desenvolvi uma compreensão aprofundada sobre como representar e manipular dados eficientemente na memória. Aprendi a trabalhar com ponteiros e memória dinâmica — elementos indispensáveis para a criação de estruturas flexíveis como listas encadeadas, que se ajustam à necessidade de armazenamento em tempo real.

A prática com pilhas e filas demonstrou como a escolha correta da estrutura de dados pode facilitar a resolução de determinados problemas, tornando o código mais limpo, eficiente e funcional. A cada exercício, fui aprimorando minha capacidade analítica para identificar a estrutura mais apropriada, considerando fatores como desempenho e uso de memória.

Projeto Final: Lista Encadeada em C

Durante o curso, desenvolvi um projeto completo de lista encadeada simples em linguagem C, disponível no GitHub. O sistema inclui todas as operações fundamentais para gerenciar a estrutura:

  • Inserção de elementos no início e no fim
  • Remoção de nós específicos
  • Busca de valores dentro da lista
  • Exibição sequencial dos dados
  • Liberação completa da memória utilizada

Exemplo de função para inserir no final da lista:

void inserir_fim(Lista *lista, int valor) {
  No *novo = criar_no(valor);
  if (novo == NULL) return;

  if (lista->inicio == NULL) {
    lista->inicio = novo;
    lista->fim = novo;
  } else {
    lista->fim->proximo = novo;
    lista->fim = novo;
  }
  lista->tamanho++;
}
