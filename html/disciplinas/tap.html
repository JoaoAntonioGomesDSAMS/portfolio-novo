Resumo da Disciplina

A disciplina tem como foco o aprimoramento das habilidades de desenvolvimento de software, com ênfase em boas práticas de codificação, padrões de projeto e arquitetura de sistemas. Utilizando a linguagem C#, os alunos aprendem a construir soluções robustas, escaláveis e de fácil manutenção, aplicando os princípios da Programação Orientada a Objetos (POO) e as diretrizes do SOLID. O curso também aborda o desenvolvimento de APIs com ASP.NET Core, integração com banco de dados via Entity Framework e a implementação de testes unitários.

Conteúdos Desenvolvidos

  • Boas práticas de desenvolvimento em C#
  • Princípios SOLID e sua aplicação na arquitetura de software
  • Padrões de Projeto (Design Patterns): soluções reutilizáveis para problemas recorrentes
  • Técnicas de refatoração de código e melhoria contínua
  • Arquitetura em camadas para aplicações escaláveis
  • Criação de testes unitários com xUnit
  • Integração com bancos de dados relacionais
  • Uso do Entity Framework para mapeamento objeto-relacional
  • Desenvolvimento de APIs RESTful com ASP.NET Core
  • Gerenciamento de dependências e aplicação de Injeção de Dependência

Abordagem Metodológica

A disciplina é estruturada em torno de uma abordagem prática e orientada a projetos, incluindo:

  • Aulas teóricas com demonstrações práticas em ambiente de desenvolvimento
  • Implementação de projetos reais utilizando C# e ASP.NET Core
  • Aplicação dos Design Patterns em cenários concretos
  • Refatoração de códigos legados com base em boas práticas
  • Exercícios individuais e atividades colaborativas em grupo
  • Criação e documentação de APIs, com foco em padronização e desempenho

Competências Desenvolvidas

Ao final da disciplina, o aluno será capaz de:

  • Aplicar boas práticas de codificação e os princípios SOLID na construção de sistemas
  • Utilizar padrões de projeto para resolver problemas recorrentes de forma elegante e eficiente
  • Estruturar sistemas com arquitetura em camadas, favorecendo manutenção e escalabilidade
  • Desenvolver APIs RESTful com ASP.NET Core e integrar sistemas com bancos de dados
  • Utilizar o Entity Framework para abstração e persistência de dados
  • Criar e aplicar testes unitários com foco em qualidade e confiabilidade do código
  • Trabalhar com injeção de dependência e gerenciamento de componentes em aplicações modernas
