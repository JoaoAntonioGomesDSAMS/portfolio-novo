Resumo da Disciplina

A disciplina aborda os fundamentos essenciais dos bancos de dados relacionais, com foco no uso prático da linguagem SQL para a criação, manipulação e segurança das estruturas de dados. Os conteúdos exploram desde os comandos básicos até a implementação de elementos reutilizáveis e estratégias de controle de acesso, promovendo o uso consciente e eficiente dos recursos de um SGBD (Sistema de Gerenciamento de Banco de Dados).

Conteúdos Trabalhados

  • Fundamentos e arquitetura de bancos de dados relacionais e não relacionais
  • Manipulação de dados com comandos SQL
  • Controle de permissões com DCL (GRANT, REVOKE)
  • Criação e utilização de views (visões) para organização de dados
  • Criação de funções definidas pelo usuário (user-defined functions)
  • Execuções práticas entre funções
  • Uso de funções personalizadas em consultas
  • Boas práticas de segurança, integridade e organização de dados

Exemplo de View:

CREATE VIEW ExemploView AS
  SELECT
    p.codigo_produto AS Codigo,
    p.descricao_produto AS Descricao,
    i.quantidade AS Quantidade,
    (i.quantidade * p.val_unit) AS Total
  FROM
    Item_do_pedido i
  INNER JOIN Produto p
    ON i.codigo_produto = p.codigo_produto;

Metodologia Aplicada

As aulas são conduzidas de forma dinâmica, integrando exposições teóricas com atividades práticas. Os alunos utilizam sistemas como MySQL e PostgreSQL para testar os conceitos aprendidos, criando procedimentos, funções e visões diretamente nos bancos de dados. A aplicação de comandos DCL também é exercitada em ambientes simulados para reforçar o controle seguro de acesso.

As atividades incluem:

  • Exercícios práticos em laboratório;
  • Projetos simples e estudos de caso inspirados em contextos reais;
  • Trabalhos e tarefas individuais;
  • Avaliações compostas por testes teóricos e análise de código SQL produzido em aula.
Exemplo de Function:
CREATE VIEW ExemploFunction
  (Codigo, Descricao, Quantidade, Total) AS
  SELECT
    i.num_pedido,
    p.codigo_produto,
    p.descricao_produto,
    i.quantidade,
    p.val_unit,
    i.quantidade * p.val_unit
  FROM Item_do_pedido i
  INNER JOIN Produto p
  ON i.codigo_produto = p.codigo_produto;

Competências Desenvolvidas

Durante o curso, os alunos adquiriram habilidades importantes para o uso profissional de bancos de dados, como:

  • Estruturação e entendimento de sistemas relacionais;
  • Automação de tarefas repetitivas com procedures e functions;
  • Criação de views para facilitar consultas e manter a organização lógica dos dados;
  • Aplicação de regras de acesso seguras por meio da DCL;
  • Aperfeiçoamento da lógica de programação dentro do contexto do SQL;
  • Visão ampliada sobre boas práticas no desenvolvimento de sistemas que envolvem persistência de dados.
